# TH Store

## How to run the application
Install the dependencies and devDependencies:

```sh
cd FinalWebDesgin
npm i
```

Create your .env file and generate the application key:

```sh
$ npm run develop
$ php artisan key:generate
```

Run migrations and start the server:

```sh
$ php artisan migrate
$ php artisan serve
```
